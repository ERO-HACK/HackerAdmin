With HakerAdmin you can find the admin page of any site.

### Install for Linux
* `apt update`
* `apt upgrade`
*  `git clone https://github.com/ERO-HACK/HackerAdmin.git`
*  `cd HackerAdmin`
*  `bash setup.sh`
*  `bash HackerAdmin.sh`
